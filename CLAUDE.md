## Project Description

SATO Printer Emulator — interprets and displays print data received in SBPL (SATO Barcode Programming Language). The project is in early development with no source code implemented yet.

## Reference Material

- `SATO_Dev_Handboook.pdf` — SBPL protocol specification and developer reference. Consult this for command syntax, print formatting, and barcode generation details when implementing the parser and renderer.
